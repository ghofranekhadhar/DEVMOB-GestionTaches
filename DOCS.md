# Documentation du projet DEVMOB-GestionTaches

## Fonctionnalités
- Authentification avec Firebase (Email/Mot de passe)
- Design moderne avec Flutter
- Logo personnalisé et interface responsive

## Architecture
- `lib/views/`: Pages de l'application
- `lib/services/`: Logique métier et services Firebase
- `lib/widgets/`: Composants UI réutilisables
